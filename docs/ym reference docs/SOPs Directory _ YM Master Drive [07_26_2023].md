**Young Muslims | SOP Committee Authority (SCA)**  
**SOP Directory**

**Summary**  
*Lore: In January 2023, five brothers rose to the call to form the SOP Committee Authority, an effort to document all the operations and processes of Young Muslims. An ambitious project indeed, but not impossible.*

This drive is a collection of documentation describing different functions and processes of Young Muslims. Use the directory and contents below to find the document you are looking for. The Master Drive contains around 90 SOPs cumulating to 250+ pages of documentation.
